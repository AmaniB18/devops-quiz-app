# DevOps Quiz App

This is a simple DevOps quiz web app created for assignment.

## File Structure

- `index.html` — HTML structure for the quiz.
- `styles.css` — Basic styling for the app.
- `app.js` — JavaScript logic to fetch and display questions.
- `data/questions.json` — Contains the quiz questions.

## Features
- Includes 20 DevOps-related questions.
- Provides multiple-choice answers.
- Displays correct/incorrect feedback.
